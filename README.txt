Промежууточная контнрольная работа. Часть 1 - приложение заметки.

Информация:
Необходимо написать проект, содержащий функционал работы с заметками.
Программа должна уметь создавать заметку, сохранять её, читать список
заметок, редактировать заметку, удалять заметку.

Задание:
Реализовать консольное приложение заметки, имеющее следующие функции:
-сохранение
-чтение
-добавление
-редактирование
-удаление

Заметка должна иметь содержать идентификатор, заголовок, тело заметки 
и дату/время создания, последнего изменения заметки.

Выбран формат json(список словарей)
