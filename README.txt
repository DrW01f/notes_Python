Промежууточная контнрольная работа. Часть 1 - приложение заметки.

Информация:
Необходимо написать проект, содержащий функционал работы с заметками.
Программа должна уметь создавать заметку, сохранять её, читать список
заметок, редактировать заметку, удалять заметку.

Задание:
Реализовать консольное приложение заметки, имеющее следующие функции:
-сохранение
-чтение
-добавление
-редактирование
-удаление

Заметка должна иметь содержать идентификатор, заголовок, тело заметки 
и дату/время создания, последнего изменения заметки. Сохранение заметок 
необходимо сделать в формате json или csv формат
Выбран формат json(список словарей)

Приложение должно запускаться без ошибок, должно уметь сохранять данные
в файл, уметь читать данные из файла, делать выборку по дате, выводить на
экран выбранную запись, выводить на экран весь список записок, добавлять
записку, редактировать ее и удалять.

Реализация без использования ООП
